# 🍔 Food Delivery App
A full-stack food ordering web app where users can browse items, login, and manage their cart.

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## ✨ Features
- User Signup & Login
- Browse Food Items
- Add to Cart
- Image Upload (Cloudinary)
- Secure Authentication

## 📁 Project Structure
food-app/
├── frontend/
├── backend/

## ⚙️ How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start

## 🔑 Environment Variables

Create a .env file in backend:

PORT=5000  
MONGO_URI=your_mongo_url  
JWT_SECRET=your_secret
